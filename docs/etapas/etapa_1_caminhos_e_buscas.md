# Etapa 1: Caminhos e Buscas

## O que vamos aprender
- O módulo `pathlib` do Python.
- A diferença entre caminhos relativos e absolutos.
- Navegação e listagem de diretórios pelo sistema operacional.
- Validação de arquivos (verificando se existem, se são pastas ou arquivos).

## Onde entra no sistema
- **Arquivo Alvo:** `src/services/scanner.py`
- **Responsabilidade:** Este módulo será responsável por receber um caminho do usuário, validar se esse caminho realmente existe no computador e varrer a pasta para encontrar os arquivos que precisamos renomear (filtrando pela extensão desejada).
