# Etapa 2: Estruturas de Dados e POO

## O que vamos aprender
- Programação Orientada a Objetos (POO).
- Criação e uso de `dataclasses` no Python.
- Criação de exceções (Exceptions) personalizadas para lidar com erros de forma limpa e descritiva.

## Onde entra no sistema
- **Arquivos Alvos:** `src/core/models.py` e `src/core/exceptions.py`
- **Responsabilidade:** Criaremos as "fôrmas" dos dados que transitam pelo nosso sistema (por exemplo, um objeto representando um arquivo a ser renomeado, com seu nome antigo e nome novo) e nossos tipos de erros específicos (ex: `DirectoryNotFound`, `InvalidExtension`).
